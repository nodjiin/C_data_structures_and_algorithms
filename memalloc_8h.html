allocate a new memory block of requested size and then initialize it to 0.

Note
failing to allocate the requested memory will result in the program to exit with a MEMORY_EXHAUSTED code.
Parameters
[in]countnumber of objects to allocate. 0 is not considered as a valid input and will cause the program to exit with an INVALID_INPUT code.
[in]sizebytes necessary to contain one of the allocated objects. 0 is not considered as a valid input and will cause the program to exit with an INVALID_INPUT code.
Returns
a pointer to the newly allocated memory.
