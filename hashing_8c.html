uint32_t hash_djb2 ( const unsigned char *  string)

hash the given string using Daniel Julius Bernstein's hash function.

Parameters
[in]stringpointer to a NUL terminated character array.
Returns
the hashed value.
Note
this hashing function is based on Linear congruential generator, and follows the formula H(n) = (a*H(n - 1) + c)mod m, where: a is 33. In the code belove Bernstein speed up the multiplication with bit manipulation: (hash << 5L) + hash. c is the value of character in position n inside the string. m is 32. We are working with an unsigned long (32 bits of size), an overflow will naturally result in a modulo 32 operation.
