Detailed Description

graph data structure.

Note
This implementation uses an adjacency list to store the neighbors of each vertex. The array containing the adjacency list doesn't resize after the initial construction, the x and y vertex values used in the insertion of an edge must therefore respect the maximum vertex size given during graph construction. This is not an hypergraph, edges connecting more than 2 vertices are not supported.
